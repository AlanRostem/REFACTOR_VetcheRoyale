Class: CClient

CClient()

Class representation of the client. Holds input callbacks and manages socket events. An instance of this class can be found in the update methods as parameters in the UIElement, CEntity, CEntityManager, InputListener and EntitySnapshotBuffer instances and as the clientRef member in Scene.
